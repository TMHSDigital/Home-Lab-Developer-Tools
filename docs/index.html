Installation

Installation steps
  1. Clone the repository
  2. Open the folder in Cursor IDE
  3. Install MCP server: cd mcp-server && npm install
  4. Configure SSH access to your Raspberry Pi / home lab nodes
  5. Start using the AI skills and rules
